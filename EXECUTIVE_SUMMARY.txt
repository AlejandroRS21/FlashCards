PROJECT OVERVIEW
================

FlashCards is a fully functional Android application that enables users to
import flashcards from CSV files, organize them into thematic decks, and
study using an intuitive gesture-based interface with smooth animations.

The application implements persistent local data storage, responsive UI design,
and comprehensive gesture support for an optimal user experience.
