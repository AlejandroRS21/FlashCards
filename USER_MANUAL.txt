USER MANUAL - FlashCards v1.3
==============================

GETTING STARTED
===============

System Requirements:
- Android 13 or later
- Minimum 100MB free storage
- 2GB RAM (recommended)
- Portrait orientation

Installation:
1. Download FlashCards APK file
2. Open file manager on device
3. Navigate to Downloads folder
4. Tap FlashCards APK
5. Tap "Install" when prompted
6. Wait for installation to complete
7. Tap "Open" to launch

First Launch:
1. App starts on Dashboard screen
2. Review statistics display
3. No decks available yet
4. Click "Library" to import first deck

MAIN SCREEN - DASHBOARD
=======================

Dashboard Components:

Top Section:
- User greeting (Good morning/afternoon/evening)
- Notification icon
- User profile avatar

Statistics Display:
- Study streak: Consecutive days studied
- Mastered cards: Total cards completed
- Daily goal progress: Percentage towards goal

Action Buttons:
- "Start Daily Review": Begin study session
- "Library": Access import screen

Recent Decks Section:
- Shows recently accessed decks
- Click any deck to study
- Cards displayed in scrollable list

All Decks Section:
- Complete list of imported decks
- Horizontal scrolling list
- Click deck to open study screen

IMPORTING DECKS
===============

Step 1: Select File

1. From Dashboard, click "Library" button
2. ImportScreen appears with "Select File" view
3. Click "Browse Files" button
4. File browser opens
5. Navigate to CSV file location
6. Select desired CSV file
7. Filename appears in selected area

File Requirements:
- Format: CSV (Comma Separated Values)
- Extension: .csv (must be lowercase)
- Encoding: UTF-8
- Minimum columns: 2 (question, answer)
- Content: Text only (no special formatting)

CSV Format Example:
---
Question 1,Answer 1
Question 2,Answer 2
"Question, with comma","Answer, with comma"
---

Step 2: Enter Deck Name

1. File selected, automatic name generated from filename
2. Click name field to edit
3. Type desired deck name (1-50 characters recommended)
4. Name must be unique
5. Clear any auto-filled text first if needed

Step 3: Review Cards

1. Click "Process CSV" to parse file
2. Loading indicator appears
3. Parsed cards displayed in preview table
4. Columns: Question | Answer
5. Scroll through preview
6. Verify data accuracy

Step 4: Create Deck

1. Review preview complete
2. Click "Create Deck" button
3. Deck is saved to local storage
4. Return to Dashboard automatically
5. New deck appears in lists

Error Handling:

Invalid File:
- Error message: "Please select a valid CSV file"
- Solution: Ensure file has .csv extension

Empty File:
- Error message: "File contains no valid cards"
- Solution: Verify CSV has at least 2 columns and 1 row

Parse Error:
- Error message: "Error parsing file: [details]"
- Solution: Check file format and encoding

STUDYING CARDS
==============

Starting Study Session:

Option 1: From Recent Decks
1. Dashboard visible
2. Locate deck in "Recent Decks" section
3. Tap deck card
4. Study screen opens

Option 2: From All Decks
1. Dashboard visible
2. Scroll "All Decks" section
3. Find desired deck
4. Tap deck card
5. Study screen opens

Study Screen Layout:

Top Bar:
- Close button (X): Exit to Dashboard
- Deck name: Current deck title
- Edit button (pencil): Edit deck name
- Stats: Current position (e.g., "1/10 cards")

Progress Bar:
- Visual progress through deck
- Updates with each card
- Shows percentage complete

Card Display:
- Large centered card
- Question visible initially
- Pastel background color
- Text centered and readable
- Multiple colors rotate through deck

Navigation Instructions:
- "Tap to see answer"
- "Swipe left for next"
- "Swipe right for previous"

Bottom Controls:
- Previous button (if not first card)
- Next button (if not last card)
- Buttons disabled at start/end

Studying Techniques:

Method 1: Swipe Navigation

Right Swipe:
1. Touch and drag card to right (>100px)
2. Current card animates out
3. Previous card slides in
4. Card automatically flips to front
5. Button becomes unavailable if at start

Left Swipe:
1. Touch and drag card to left (>100px)
2. Current card animates out
3. Next card slides in
4. Card automatically flips to front
5. Button becomes unavailable if at end

Method 2: Button Navigation

Previous Button:
1. Click "Previous" at bottom
2. Move to previous card
3. Card flips to question side
4. Disabled when on first card

Next Button:
1. Click "Next" at bottom
2. Move to next card
3. Card flips to question side
4. Disabled when on last card

Method 3: Flip Cards

1. Card displays question
2. Tap anywhere on card (single tap)
3. Card smoothly rotates 180 degrees
4. Answer appears (takes 500ms)
5. Tap again to flip back

Multicolor Feature:
- Each card has pastel background
- 6 different colors used
- Colors rotate through deck
- Helps with visual differentiation

MANAGING DECKS
==============

Edit Deck Name:

1. Open deck in study screen
2. Click pencil (edit) icon next to deck name
3. Dialog box appears
4. Current name highlighted
5. Type new name (clear old text first)
6. Click "Save" button
7. Name updates immediately
8. Change reflected in Dashboard
9. Data auto-saved to device storage

Note: Editing only changes name, not cards.

Delete Deck:

Decks cannot be deleted from study screen.
To delete (if feature implemented):
1. Return to Dashboard
2. Long press deck card
3. Select "Delete" from menu
4. Confirm deletion
5. Deck removed from all lists

Import Additional Decks:

1. From Dashboard, click "Library"
2. Follow import steps as before
3. Can import multiple decks
4. All decks stored independently
5. Switch between decks anytime

Deck Limits:
- No maximum number of decks
- No maximum cards per deck
- Limited by device storage only

TIPS AND BEST PRACTICES
=======================

Effective Studying:

1. Study Regularly:
   - Consistent daily study sessions
   - 5-10 minutes minimum recommended
   - Build learning streaks

2. Review Frequently:
   - Multiple passes through deck
   - Focus on difficult cards
   - Reinforce memory

3. Use Multiple Methods:
   - Mix swipe and button navigation
   - Vary study rhythm
   - Prevent monotony

4. Create Quality Content:
   - Clear, concise questions
   - Accurate answers
   - Avoid very long text

5. Organize by Topic:
   - Create focused decks
   - One topic per deck
   - Easier to track progress

CSV File Tips:

1. Excel to CSV:
   - Open Excel file
   - File > Save As
   - Format: CSV (UTF-8)
   - Save in Downloads folder

2. Google Sheets to CSV:
   - File > Download > CSV
   - Specify column order
   - Download to device
   - Import through app

3. Text Editor CSV:
   - Open text editor
   - Enter data as shown above
   - Save with .csv extension
   - UTF-8 encoding required

4. Special Characters:
   - Enclose in quotation marks
   - Commas must have quotes
   - Example: "Paris, France"

TROUBLESHOOTING
===============

Issue: App won't open

Solution:
1. Check Android version (13+)
2. Restart device
3. Clear app cache: Settings > Apps > FlashCards > Storage > Clear Cache
4. Uninstall and reinstall app
5. Ensure 100MB+ free storage

Issue: Cannot select file

Solution:
1. Verify file has .csv extension (lowercase)
2. Move file to Downloads folder
3. File name should not contain special characters
4. Try different file location
5. Check file permissions (readable)

Issue: Cards appear empty

Solution:
1. Check CSV format in file
2. Ensure columns are separated by commas
3. Verify no hidden characters
4. Re-import with valid CSV
5. Check device storage space

Issue: Cannot edit deck name

Solution:
1. Ensure in study screen (not dashboard)
2. Look for pencil icon next to title
3. Wait for dialog to fully appear
4. Check name field is active
5. Ensure name not blank before saving

Issue: App crashes during import

Solution:
1. Restart app
2. Try smaller CSV file (test import)
3. Check file encoding (must be UTF-8)
4. Remove special characters from deck name
5. Free up device storage space

Issue: Data not saving

Solution:
1. Check device storage (needs 100MB+)
2. Ensure app has storage permissions
3. Restart device
4. Try creating new deck
5. Check if deck appears after closing/reopening app

Issue: Swipe not working

Solution:
1. Ensure swipe > 100 pixels
2. Swipe should be side-to-side (horizontal)
3. Avoid swiping at card edges
4. Try button navigation instead
5. Restart app if frozen

Performance Issues:

Slow CSV Import:
1. Break large file into smaller files
2. Remove unnecessary columns
3. Try different file location
4. Clear app cache
5. Restart device

Laggy Study Screen:
1. Close other apps
2. Reduce animations (restart app)
3. Check device storage space
4. Update Android system
5. Clear device cache

Storage Issues:

Clear Space:
1. Delete unused decks (if feature available)
2. Clear app cache: Settings > Apps > FlashCards
3. Delete old CSV files
4. Clear device downloads folder
5. Uninstall unused apps

DATA BACKUP
===========

Automatic Backup:

Android Device:
1. Settings > Google > Manage your Google Account
2. Tap "Backup" tab
3. FlashCards data eligible for backup
4. Enable "Backup by Google One"
5. Data backs up automatically

Manual Backup:

CSV Export (if implemented):
1. Study screen > Menu > Export
2. Choose location to save
3. CSV file created with cards
4. Can email or share file
5. Can re-import later

Cloud Storage (if implemented):
1. Study screen > Menu > Cloud Sync
2. Sign in with account
3. Decks synced to cloud
4. Access from multiple devices
5. Automatic updates

DATA PRIVACY
============

Local Storage:
- All data stored on device only
- No internet transmission
- No cloud synchronization required
- No account creation needed
- Completely private study sessions

Permissions Requested:
- READ_EXTERNAL_STORAGE: For CSV file selection
- No other permissions required
- No camera, microphone, or location access
- No contact or messaging access

Data Deletion:
- Uninstall app to remove all data
- Or use "Clear Data" in app settings
- No recovery after deletion
