## ⚡ Inicio Rápido (5 minutos)

### ¿Qué es esta skill?
Una **colección de patrones y componentes de diseño** para construir apps Android hermosas, accesibles y responsivas usando Jetpack Compose.

### ¿Qué obtengo?
- ✅ 50+ patrones de diseño probados
- ✅ 20+ componentes reutilizables
- ✅ Guías de accesibilidad WCAG AA
- ✅ 50+ ejemplos de código
- ✅ Implementación lista para FlashCards

### ¿Cómo empiezo?

**Opción A (5 min)**: Lee este archivo + SKILL_QUICK_REFERENCE.md  
**Opción B (30 min)**: Lee FLASHCARDS_IMPLEMENTATION.md  
**Opción C (1h)**: Lee ANDROID_MOBILE_DESIGN_SKILL.md completo
